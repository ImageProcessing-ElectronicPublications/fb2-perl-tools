# fb2-perl-tools

Perl tools for manipulate FB2 books

## fb2optimage.sh

Lossless optimize images in fb2 books.

USAGE: ./fb2optimage.sh file.fb2

DEPENDS: dash, coreutils, imagemagick, jpegrescan-perl, gifsicle, optipng, advancecomp
